# 육진대 맵 (Yukjindae Map) 기획안

> 육진대 아빠들이 직접 다녀온 **아빠와 아이가 갈만한 곳** 전국 지역 베스트 지도 웹앱

---

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 육진대 맵 |
| 타겟 | 육진대 멤버 (300+ 아빠 크리에이터) |
| 플랫폼 | 웹앱 (PWA 고려) |
| 배포 | Cloudflare Pages |
| 데이터 관리 | Notion DB |
| 지도 API | 네이버 Maps API (Dynamic Map + Geocoding) |

---

## 2. 핵심 컨셉

- 육진대 아빠 멤버들이 **직접 다녀온 장소**만 등록
- 앱스토어 심사 없이 **링크 공유**로 바로 사용
- 멤버들이 Notion에서 **직접 장소 추가** 가능
- **지역별 카테고리**로 빠르게 탐색

---

## 3. 기능 요구사항

### 3-1. 홈 화면
- [ ] 히어로 배너 (육진대 소개 + 통계)
- [ ] 장소 검색창
- [ ] 전국 지역 카테고리 그리드 (8개 지역)
- [ ] 카테고리 태그 필터 (자연/공원, 실내놀이, 맛집, 카페, 체험/문화, 스포츠, 무료)
- [ ] 추천 장소 카드 리스트

### 3-2. 지도 화면
- [ ] 네이버 지도 전체 화면
- [ ] 지역별 장소 핀 표시
- [ ] 핀 클러스터링 (핀 많을 때 숫자로 묶기)
- [ ] 핀 클릭 시 장소 카드 팝업
- [ ] 현재 내 위치 표시
- [ ] 지역 클릭 시 줌인

### 3-3. 장소 카드
- [ ] 장소 이미지
- [ ] 장소명, 지역, 카테고리 태그
- [ ] 추천 이유 (한 줄 요약)
- [ ] 운영시간, 입장료
- [ ] 거리 표시 (현재 위치 기준)
- [ ] 평점 + 리뷰 수
- [ ] 좋아요 버튼
- [ ] 카카오맵/네이버 지도 길찾기 연동

### 3-4. 장소 상세 페이지
- [ ] 이미지 갤러리
- [ ] 상세 정보 (주소, 운영시간, 입장료, 주차)
- [ ] 추천 이유 (육진대 멤버 후기)
- [ ] 근처 맛집/카페 추천
- [ ] 지도에서 위치 보기
- [ ] 공유하기 버튼

### 3-5. 찜 목록
- [ ] 좋아요한 장소 모아보기
- [ ] 로컬 스토리지 저장

### 3-6. 후기 제보 (MVP 이후)
- [ ] 장소 제보 폼
- [ ] Notion DB 자동 저장
- [ ] 관리자 승인 후 지도 표시

---

## 4. 데이터 구조 (Notion DB)

### 장소 DB 컬럼

| 컬럼명 | 타입 | 설명 |
|--------|------|------|
| 장소명 | Title | 장소 이름 |
| 지역 | Select | 서울/경기북부/인천/강원도/충청도/전라도/경상도/제주 |
| 카테고리 | Multi-select | 자연·공원/실내놀이/맛집/카페/체험·문화/스포츠 |
| 주소 | Text | 도로명 주소 |
| 위도 | Number | 좌표 (Geocoding API로 자동 변환 가능) |
| 경도 | Number | 좌표 |
| 사진 | Files | 장소 이미지 (멤버 직촬 권장) |
| 추천이유 | Text | 한 줄 추천 이유 |
| 운영시간 | Text | 예: 10:00~18:00 (월요일 휴무) |
| 입장료 | Text | 예: 무료 / 성인 10,000원 |
| 주차 | Select | 가능/불가/유료 |
| 등록자 | Text | 육진대 멤버 닉네임 |
| 공개여부 | Checkbox | 승인된 장소만 표시 |

## 6. 개발 단계

### MVP (1단계)
- 홈 화면 + 지역 카테고리
- 네이버 지도 연동 + 핀 표시
- Notion DB 장소 데이터 연동
- 장소 카드 리스트
- Cloudflare Pages 배포

### 2단계
- 현재 위치 기반 거리 표시
- 카테고리 필터
- 장소 상세 페이지
- 길찾기 연동

### 3단계
- 좋아요/찜 기능
- 장소 제보 기능
- PWA 적용 (홈 화면 아이콘 추가)

## 7. 디자인 방향

### 컬러 시스템
> 육진대 로고(`육진대첫번째로고02.jpg`)의 색감을 기준으로 전체 디자인 통일

| 역할 | 컬러 | 설명 |
|------|------|------|
| Primary (메인) | `#1A2F6B` | 로고 다크 네이비 |
| Secondary | `#2563EB` | 로고 미디엄 블루 |
| Accent | `#4A90D9` | 로고 라이트 블루 |
| Background | `#F0F4FF` | 연한 블루 배경 |
| Text | `#0D1B3E` | 다크 네이비 텍스트 |
| White | `#FFFFFF` | 카드 배경 |

### 디자인 원칙
- 전체 톤: **네이비 + 블루 계열** (로고와 동일한 색감)
- 카무플라주 패턴은 포인트 요소로 활용 가능
- 아빠+아이 실루엣 모티브 UI 요소에 활용
- 모바일 퍼스트 (최대 너비 480px)
- 하단 탭바 네비게이션
- 따뜻하되 강인한 느낌 (육진대 = 육아에 진심인 대디)
